# Currency Converter

A lightweight currency converter built with vanilla HTML, CSS, and JavaScript.

## Features

- Clean, responsive design
- No frameworks or dependencies
- Easy to modify and extend

## How to Use

1. Clone this repository
2. Open `index.html` in your browser
3. Select currencies from the dropdown menus
4. Enter amount to convert
5. View converted amount instantly

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
